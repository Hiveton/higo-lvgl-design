# 产品方向

## 目标

建设一个浏览器内运行的 LVGL 在线编辑器，让嵌入式 UI 团队可以从视觉搭建、资源管理、模拟预览到 LVGL C 代码交付完成闭环。

## 适用范围

本文档定义产品定位、目标用户、第一版价值、路线规划和明确不做的内容。所有功能拆分和架构实现应服务这里定义的方向。

## 非目标

第一版不做多人协作、不做真实硬件烧录、不做完整云端 SDK 固件编译、不做商业素材市场，也不复制 SquareLine Studio 的界面或品牌。

## 产品定位

LVGL Online Editor 是一个面向嵌入式显示项目的在线 IDE。它不是营销展示页，也不是通用设计工具，而是一个面向工程交付的开发工具：控件、图层、属性、事件、资源、预览和代码导出都必须围绕 LVGL 项目落地。

## 目标用户

- 嵌入式 UI 工程师：需要快速搭建 LVGL 界面并导出 C 代码。
- 硬件产品团队：需要在没有完整桌面开发环境的情况下预览屏幕 UI。
- LVGL 项目开发者：需要维护多个屏幕、资源和控件样式。
- 小型团队或方案商：需要基于云项目沉淀可复用 UI 模板。

## 第一版核心价值

- 不安装桌面 IDE 即可创建和保存 LVGL UI 项目。
- 通过拖拽和 Inspector 编辑常用 LVGL 控件。
- 使用浏览器内 WASM 模拟器快速预览设备界面。
- 导出稳定、可读、可 diff 的 LVGL C 代码。
- 用 ProjectDoc 作为前端、后端、模拟器、代码生成的统一协议。

## 产品路线

### V1：完整 IDE 第一版

- 单人账号与云项目。
- 多 screen 管理。
- Widgets palette、Layers tree、Canvas、Inspector。
- 图片资源上传和 image widget 引用。
- WASM 模拟器预览。
- LVGL `v8.3.x` C 代码 zip 导出。
- autosave、undo、redo、日志面板。

### V1.5：工程效率增强

- 项目模板和设备模板。
- 主题系统与样式复用。
- 更多 LVGL 控件和复杂属性。
- 图片资源压缩和字体资源处理。
- 代码生成配置项，例如命名前缀、是否拆分文件、是否生成 callback stub。

### V2：团队和硬件闭环

- 团队空间、权限和项目分享。
- 版本对比、分支或快照恢复。
- 插件系统。
- 云编译或本地 SDK agent。
- 设备下载、日志回传和硬件预览。

## 成功标准

- 新用户能在 10 分钟内创建项目、拖入控件、修改文本和样式、预览并导出 C 代码。
- 导出的 C 代码能被 LVGL `v8.3.x` 工程直接集成或少量适配后集成。
- 编辑器交互符合专业 IDE 预期：状态明确、无明显卡顿、可撤销、可保存、可恢复。
- 第一版项目文档能让前后端与 codegen 工程师并行开发。
